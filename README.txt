AT&T API Samples - Speech app 1
 ------------------------------

1.Install the app
navigator to /Att-Sample/tree/master/RestFull/Speech/Ruby/app1,
 run "bundle install --path vendor"
 
4. Running the application

 bundle exec ruby ./speech.rb

Your application becomes available in a web browser, so you may visit: http://localhost:4567/ to see it working.

You can copy your test .wav file into /Att-Sample/tree/master/RestFull/Speech/Ruby/app1/public/audio/ to test it
